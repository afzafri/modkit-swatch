# Getting Started

How to set up ModKit Swatch for local development.

## Prerequisites

- Node.js 18+
- npm

## Installation

```bash
git clone https://github.com/afzafri/modkit-swatch.git
cd modkit-swatch
npm install
```

## Development Server

```bash
npm run dev
```

Open `http://localhost:3000`.

## Build for Production

```bash
npm run build
npm run start
```

## Linting

```bash
npm run lint
```

## Project Commands

| Command           | Description              |
| ----------------- | ------------------------ |
| `npm run dev`     | Start development server |
| `npm run build`   | Production build         |
| `npm run start`   | Start production server  |
| `npm run lint`    | Run ESLint               |
